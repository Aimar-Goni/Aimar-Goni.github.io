Vorpal Engine

C++OpenGL


A proprietary engine written in C++ that uses OpenGL, and integrates VR, portals, and advanced rendering techniques. The engine was developed by Carlos Cordero and myself during our last year of an HND in Computer Science, for the subject of Engine Programming (8 months | October 2023 - May 2024).


What I did

In this project, developed the logging system, managed the user input, implemented the user interface system, implemented an job system with multithreading, an Entity Component System, the camera system, application window management, rendering, billboards, scene management and advanced techniques like SSAO
