Vorpal Engine

C++ OpenGL VR

Custom game engine with portals

Engine Overview

A proprietary engine written in C++ that uses OpenGL, and integrates VR, portals, and advanced rendering techniques. The engine was developed by Carlos Cordero and myself during our last year of an HND in Computer Science, for the subject of Engine Programming (8 months | October 2023 - May 2024).

Technical Features

  • Portal Rendering - Recursive scene rendering with occlusion culling
  • VR Integration - Dual-eye rendering with OpenVR SDK support
  • ECS Architecture - Entity Component System for game objects
  • Advanced Rendering - Deferred shading, SSAO, shadow mapping

My Contributions

  • Developed the logging system and user input management
  • Implemented the UI system and multithreaded job system
  • Created the Entity Component System architecture
  • Built camera system and application window management
  • Implemented advanced graphics techniques (SSAO, billboards)
